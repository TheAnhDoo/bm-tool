# BM Invite Tool

Facebook automation system for managing VIA and BM profiles with advanced fingerprinting capabilities.

## Features

- Profile management (VIA and BM profiles)
- Advanced device fingerprinting
- Automated browser control with Puppeteer
- Secure credential storage with encryption
- Real-time dashboard with statistics
- Batch operations support

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Backend**: Electron + Fastify + Prisma
- **Database**: SQLite
- **Automation**: Puppeteer with stealth plugins
- **UI**: Shadcn UI components

## Setup

1. Install dependencies:
```bash
cd bminvite
npm install
```

2. Initialize database:
```bash
npx prisma migrate dev
```

3. Run development:
```bash
npm run dev
```
